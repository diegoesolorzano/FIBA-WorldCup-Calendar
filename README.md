# FIBA World Cup 2027 Americas Qualifiers - Calendar

Auto-updating `.ics` calendars for the FIBA Basketball World Cup 2027 Americas Qualifiers. Subscribe to your team's calendar and get game times, scores, and results updated automatically.

**[View Calendar](https://diegoesolorzano.github.io/FIBA-WorldCup-Calendar/)**

## How it works

1. Scrapes game data from the FIBA website daily
2. Generates `.ics` calendar files for each of the 16 Americas teams
3. Deploys to GitHub Pages so calendars are subscribable from any calendar app

## Teams

Argentina, Bahamas, Brazil, Canada, Chile, Colombia, Cuba, Dominican Republic, Jamaica, Mexico, Nicaragua, Panama, Puerto Rico, Uruguay, USA, Venezuela.

## Subscribe

Pick your team at the [landing page](https://diegoesolorzano.github.io/FIBA-WorldCup-Calendar/) or use a direct URL:

```
https://diegoesolorzano.github.io/FIBA-WorldCup-Calendar/calendars/ven.ics
```

Replace `ven` with your team's code. Lowercase three-letter country codes.

**Google Calendar:** Settings > Add calendar > From URL > paste the link.

**Apple Calendar:** File > New Subscription > paste the link.

## Data source

All game data is scraped from [FIBA Basketball](https://www.fiba.basketball). Updated daily via GitHub Actions.
